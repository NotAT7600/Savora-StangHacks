# PantryAI Redesign - Dark UI Overhaul

## Goal
Transform PantryAI into a polished, dark-themed MVP with shadcn/Linear/Vercel-inspired design.

## Design Priorities
1. Consistent dark theme (#0a0a0a backgrounds)
2. Text-first Community Feed (no images)
3. Professional polish (not hackathon rough)
4. Real data only (no fake content)
5. Proper empty states
